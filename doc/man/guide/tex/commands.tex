\chapter{かな漢字変換ユーティリティ}
\label{4かな漢字変換ユーティリティ}

本システムではかな漢字変換辞書のメンテナンスやかな漢字変換サーバのメン
テナンスを行うための種々のユーティリティを提供しています。ユーティリティ
は辞書をメンテナンスするものとその他の2つに大きく分けられ、辞書をメン
テナンスするものに関しては、かな漢字変換サーバを介して辞書のメンテナン
スが可能なものと、辞書ファイルが存在するマシン上でなければメンテナンス
ができないものがあります。

この章ではこれらのユーティリティツールに関する説明をマニュアルページの
形で示します。

辞書の変換や、新しい辞書の作成などのために以下のようなユーティリティを
提供しています。

\begin{nquote}{2em}
\begin{namelist}{cannaserverxxxx}
\item [mkromdic] テキスト形式のローマ字かな変換テーブルをバイナリ形式に変換する。
\item [mkbindic] テキスト形式の辞書をバイナリ形式の辞書に変換する。
\item [dpbindic] バイナリ形式の辞書をテキスト形式の辞書に変換する。
\item [dpromdic] バイナリ形式のローマ字かな変換テーブルをテキスト形式に変換する。
\item [dicar] バイナリ形式の辞書のアーカイブを行う。
\end{namelist}
\end{nquote}

以下はかな漢字変換サーバを介して辞書のメンテナンスが可能なツールです。

\begin{nquote}{2em}
\begin{namelist}{cannaserverxxxx}
\item [mkdic] 辞書の作成／アップロードを行う。
\item [lsdic] 辞書の一覧を出力する。
\item [rmdic] 辞書を削除する。
\item [mvdic] 辞書の名前を変更する。
\item [cpdic] 辞書をコピーする。
\item [catdic] 辞書のダウンロードを行う。
\item [addwords] 単語の一括登録を行う。
\item [delwords] 単語の一括削除を行う。
\item [chmoddic] 辞書のアクセス権の変更を行う。
\end{namelist}
\end{nquote}

また、初期状態のチェックや、かな漢字変換サーバに関する以下のようなコマ
ンドを提供しています。

\begin{nquote}{2em}
\begin{namelist}{cannaserverxxxx}
\item [cshost] かな漢字変換システムのサーバ・アクセス制御状態を表示する。
\item [cannacheck] 日本語入力システム関連情報を表示する。
\item [cannakill] かな漢字変換サーバcannaserverを終了させる。
\item [cannaserver] かな漢字変換のサービスを提供する。
\item [cannastat] かな漢字変換サーバの情報を表示する。
\end{namelist}
\end{nquote}
%
\par
カスタマイズファイルを自動的に作成修正するツールを提供しています。
%
\begin{nquote}{2em}
\begin{namelist}{cannaserverxxxx}
\item [ccustom] 日本語入力システムのカスタマイズをする。
\end{namelist}
\end{nquote}
%
\par
かな漢字変換システムWnnとのテキスト辞書互換ツールを提供しています。
%
\begin{nquote}{2em}
\begin{namelist}{cannaserverxxxx}
\item [ctow] 本日本語入力システムのテキスト形式の辞書をWnnのテキスト形式の辞書に変換する。
\item [wtoc] Wnnのテキスト形式の辞書を本日本語入力システムのテキスト形式の辞書に変換する。
\item [splitword] 複数のテキスト形式の辞書を１つにして、１行１品詞１候補にする。
\end{namelist}
\end{nquote}

端末での日本語環境設定のために以下のコマンドを提供しています。

\begin{nquote}{2em}
\begin{namelist}{cannaserverxxxx}
\item [canuum] 端末でのかな漢字変換を行う。
\end{namelist}
\end{nquote}
